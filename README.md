# AI-tool-usage-analysis-india
📌 Project Overview

This project analyzes how Indian college students use AI tools and examines their impact on academic performance across different streams, devices, and states.

🛠 Tools Used
Excel
Power BI

📂 Dataset
Source: Kaggle
Size: 3,614 students
Features: 16 columns

🔍 Data Cleaning
Handled 1,614 missing values in the State column
Removed duplicate records
Converted CSV to XLSX format

📊 Analysis
Created 6 Pivot Tables
Built KPI sheet
Developed 3 charts

📈 Dashboard
Built 9 interactive visuals
Added state filter slicer
Used KPI cards, bar charts, donut charts, pie chart

💡 Key Insights
ChatGPT is the most used AI tool among students
Science students use AI more than Engineering students
Academic impact is almost evenly split
52% professors don’t allow AI usage
Willingness to pay is evenly divided
Laptops are the most preferred device

🎯 Outcome

This project highlights how AI adoption is influencing student learning behavior and demonstrates practical data analysis and visualization skills.
